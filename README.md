# sdn-tpot



## Getting Started


### Prerequisites


```
modprobe openvswitch
apt-get install docker-compose
cd sdn-tpot/main/
cp crontab /etc/crontab
docker-compose up -d
pip install --upgrade python-iptables
pip install docker
```

### Installing

```
/bin/bash ./setup_sdn_tpot
```
